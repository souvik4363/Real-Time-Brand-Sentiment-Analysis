# Apple Brand Sentiment Analysis Dashboard

## Project Overview

This project collects Apple-related news articles using NewsAPI, performs sentiment analysis using TextBlob, and visualizes the results in Power BI.

## Tools & Technologies

- Python
- Pandas
- Requests
- TextBlob
- NewsAPI
- Power BI

## Project Workflow

1. Extract Apple news articles using NewsAPI.
2. Store data in CSV format.
3. Perform sentiment analysis using TextBlob.
4. Classify articles as Positive, Neutral, or Negative.
5. Build an interactive dashboard in Power BI.
6. Generate insights and business conclusions.

## Results

- Total Articles Analyzed: 79
- Positive Articles: 33 (41.77%)
- Neutral Articles: 39 (49.37%)
- Negative Articles: 7 (8.86%)

## Dashboard Features

- Total Articles KPI
- Positive Articles KPI
- Neutral Articles KPI
- Negative Articles KPI
- Sentiment Distribution Pie Chart
- Articles by News Source Bar Chart
- Insights & Conclusion Page
